# LocalLibrary
- Created for the purpose of learning node, express, pug, mongoose, and mongoDB
## Build setup
```
# install dependencies
$ npm install 

# run server at localhost
$ npm start

# build project
$ npm build
```
## Tools
- Node.js
- Express
- Mongoose
- MongoDB
- Pug
